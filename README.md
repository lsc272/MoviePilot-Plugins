# SmartCollections（智能合集）

MoviePilot V2 插件：从热门 TMDB 片单、热门豆瓣豆列或手动输入的公开片单链接读取电影和剧集，按 TMDB ID 精确匹配 Emby 媒体库中已有项目，并创建或更新 Emby 合集。

当前版本：v0.5.1

## 功能

- Vue 智能合集工作台：原生集成在 MoviePilot 插件详情页，并可显示独立侧栏入口。
- 热门 TMDB 片单：支持热门/高分榜单，以及奥斯卡、金球奖、IMDb、豆瓣 Top 250、Letterboxd、Criterion、TSPDT、S&S、A24、MUBI 等公开片单；目录动态使用来源实际标题，并显示条目数量。
- 热门豆瓣豆列：直接浏览公开豆列，目录和新建合集优先采用豆列页面返回的实际标题。
- 手动链接：粘贴公开 TMDB List 或豆瓣豆列链接，自动读取合集名称。
- TMDB List：支持 TMDB v4 混合片单；未配置 v4 Token 时回退到 MoviePilot 的 TMDB v3 API Key。
- TMDB 分页：v4 与 v3 回退路径都会逐页读取，直到来源结束或达到配置的最大条目数，不再停留在默认第一页 20 条。
- 豆瓣豆列：通过 MoviePilot 官方 `MediaChain.get_tmdbinfo_by_doubanid` 转换为 TMDB ID，兼容电影和剧集，并缓存识别结果。
- 豆瓣大豆列容错：优先用 Emby 的标题、原名和年份匹配；大型豆列会并发按中外文别名查询 TMDB，较小豆列在豆瓣 ID 转换被限流时也会自动回退，剧集季度标题可映射到整部剧集。
- 豆瓣映射缓存：成功的“豆瓣 ID → TMDB ID”长期保存在 MoviePilot 插件数据中；失败结果缓存 6 小时后自动重试，避免短时间反复触发豆瓣限流。
- 共享种子缓存：插件会合并仓库内经过审核的 `assets/douban_tmdb_seed.json`，本地成功映射优先，已审核种子可替换本地临时失败项。共享映射通过 GitHub PR 审核，避免匿名自动上传造成错误映射或恶意污染。
- 缓存贡献：工作台可导出本机识别成功的映射 JSON；用户核对后通过 GitHub PR 合并至共享种子，插件不会自动上传任何映射。
- 豆列数量提示：同时展示豆瓣标称数量和公开页面实际返回数量；遇到已删除、失效或仅创建者可见的条目时会明确提示差额。
- 匹配预览：在后台解析最多 2000 个条目，页面实时显示进度，完成后发送 MoviePilot 通知；逐项展示海报、片名、年份、类型、TMDB ID、Emby 匹配和缺失原因。
- 缺失订阅：未匹配但具有 TMDB ID 的电影或剧集可单独订阅，也可在预览中一键订阅全部缺失项目；均使用 MoviePilot 系统默认订阅规则并立即搜索资源。
- 海报交互：点击影视海报可放大查看；标题列打开 TMDB，绿色 Emby 匹配列打开对应 Emby 项目页。
- Emby 匹配：优先按媒体类型与 TMDB ID 精确匹配，必要时以唯一的标题和年份结果兜底。
- 模板：可将当前预览保存为模板，并通过 JSON 导入或导出。
- 批量同步：多选公开片单或豆列后，以后台任务依次创建或更新多个 Emby 合集；页面实时展示片单内进度及总进度，离开页面不会中断，完成后发送 MoviePilot 通知，避免长任务触发网关 504。
- 合集管理：以紧凑列表记录由插件管理的合集，支持单个或定时批量重新同步，也可同时删除 Emby 合集。
- 其他合集工具：识别未由智能合集管理的 Emby 合集，支持本地 ZIP 备份、单份备份删除、合并恢复，以及强制自动备份后的批量清理；备份包含成员、元数据和压缩主海报并保留最近五份。
- 合集元数据：预览展示来源简介，同步时自动将简介写入 Emby 合集。
- 合集海报：使用多张影视海报生成倾斜海报墙，黑色遮罩从底部向上渐隐；标题采用暖金渐变、居中大字和按完整单词平衡换行。也可随时重新生成或上传自定义图片，旧版自动海报会在下次同步时升级。
- 两种更新模式：`sync` 会增删成员，使合集与片单一致；`append` 只添加新成员。
- 支持 Cron 定时同步、已管理合集独立定时批量重同步、立即运行、远程命令、运行记录和通知。
- 安全保护：来源标称数量未读完，或来源/匹配数量相比上次异常缩水时，完全同步会自动降级为只追加并在通知中说明原因；网络错误不会误删已有合集成员。

## 贡献豆瓣映射缓存

在工作台点击“导出映射缓存”，会下载仅包含本机成功识别且不含共享种子的 JSON。提交前请抽查豆瓣条目与 TMDB 类型、ID 和年份，然后将 `mappings` 合并到 [`douban_tmdb_seed.json`](plugins.v2/smartcollections/assets/douban_tmdb_seed.json)，通过 GitHub Pull Request 提交。只有审核通过的映射才会随后续插件版本分发。

## 安装

本插件是 V2 专用插件，目录应保持为：

```text
plugins.v2/smartcollections/
```

将本仓库加入 MoviePilot 插件市场，或把插件目录放入 MoviePilot 的本地插件仓库后安装。插件会直接复用 MoviePilot 中已经启用的 Emby 配置，不需要再次填写 Emby 地址和 API Key。

## 配置来源

智能合集工作台提供三种来源，可以同时使用：

1. 在“热门 TMDB 片单”中多选需要的电影或剧集榜单。
2. 在“热门豆瓣豆列”中多选需要的公开豆列。
3. 在“手动添加”中每行粘贴一个链接，例如：

```text
https://www.themoviedb.org/list/5292-tmdb-watchlist
https://www.douban.com/doulist/240962/
```

插件会先展示匹配和缺失结果；确认合集名称与条目选择后再同步到 Emby，不需要填写配置 JSON。

TMDB v4 Read Access Token 建议填写，以支持新版混合片单中的电影和剧集。Token 只保存在 MoviePilot 插件配置中。

## 使用

- 保存配置时勾选“立即运行一次”，插件会在数秒后执行。
- 启用定时同步后按 Cron 表达式运行，默认每天 04:00。
- 远程命令：`/smartcollections_sync`。
- 插件详情页包含片单目录、模板和已同步合集三个工作区；同步记录保留最近二十次。

首次同步大型豆列需要为未在 Emby 中直接命中的条目查询 TMDB，速度会稍慢；后续运行会使用插件缓存。

## 说明

This product uses the TMDB API but is not endorsed or certified by TMDB.

豆瓣网页结构和访问策略可能变化。插件仅支持无需登录即可访问的公开豆列，并遵守配置的最大读取条目数。
